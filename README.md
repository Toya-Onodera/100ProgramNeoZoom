# 100ProgramNeoZoom
オンラインチャットツールの不便な点を解消してみるプロトタイプ (100Program で使用)

### 起動方法
1. SkyWay のアクセスキーを取得する
2. `yarn` (プロジェクトのルートディレクトリ)
3. `yarn start`
